# 噤声之屋 (Hush House) - 项目优化与扩展计划

## 1. 沉浸式体验增强 (Immersive Experience)

### 视觉效果 (Visuals)
- **动态光影 (Dynamic Lighting)**: 
  - 在书页上增加跟随鼠标移动的微弱光源（如烛光效果），模拟在昏暗图书馆阅读的感觉。
  - 使用 CSS `mix-blend-mode` 实现更真实的墨水渗入纸张的效果。
- **粒子系统 (Particle System)**:
  - 在背景中增加漂浮的尘埃微粒（Dust Motes），增强古老、静谧的氛围。
  - 翻页时产生微小的纸屑或魔法火花。
- **自定义光标 (Custom Cursor)**:
  - 将鼠标光标替换为羽毛笔、放大镜或神秘符号，点击时产生墨水涟漪效果。

### 听觉反馈 (Audio)
- **环境音效 (Ambience)**:
  - 添加可开关的背景白噪音：雨声打在窗户上、壁炉燃烧声、远处钟楼的钟声。
- **交互音效 (SFX)**:
  - 翻页时的纸张摩擦声（区分轻重）。
  - 打开/关闭书籍时的皮具摩擦声。
  - 鼠标悬停在神秘符文上时的低语声（Whispers）。

## 2. 游戏化功能扩展 (Gamification & Gameplay)

### 知识收集 (Lore Collection)
- **碎片系统 (Fragments)**:
  - 允许用户“收集”书中的特定段落或关键词，存入个人的“笔记本”中。
  - 类似于《密教模拟器》中的卡牌栏，用户可以将收集到的线索拖拽组合。
- **隐藏秘密 (Hidden Secrets)**:
  - 在书页中埋藏隐形墨水文字，只有当鼠标悬停或开启特定“透镜”模式时才可见。
  - 点击特定的符文触发隐藏的支线故事或彩蛋。

### 研究面板 (Research Board)
- **线索连线 (Clue Board)**:
  - 提供一个可视化面板，展示不同书籍、实体之间的关系（基于现有的 Knowledge Graph）。
  - 允许用户手动连线，构建自己的推理图。

## 3. 技术与性能优化 (Technical & Performance)

### 移动端适配 (Mobile Adaptation)
- **手势操作 (Gestures)**:
  - 在移动端支持滑动翻页、双指缩放查看细节。
  - 优化 3D 翻页在低性能设备上的表现，必要时自动降级为 2D 切换。
- **PWA 支持**:
  - 将网站配置为 Progressive Web App，允许用户安装到桌面/手机，支持离线阅读已缓存的书籍。

### 性能优化 (Optimization)
- **资源懒加载 (Lazy Loading)**:
  - 对纹理图片、字体文件进行预加载和懒加载策略优化，减少首屏加载时间。
- **虚拟滚动 (Virtual Scrolling)**:
  - 如果书籍内容非常长，在简约模式下引入虚拟滚动以保持流畅度。

## 4. 内容管理与创作 (Content & Creation)

### 模组支持 (Modding Support)
- **自定义书籍导入**:
  - 允许用户上传 JSON/Markdown 文件，自动生成符合格式的古籍。
  - 提供一个简单的编辑器界面，所见即所得地编辑书籍内容和排版。

### 多语言支持 (i18n)
- **动态翻译**:
  - 虽然目前是中文，但可以预留架构支持英文原版对照，点击段落可切换语言。
