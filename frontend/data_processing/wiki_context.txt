# 司辰
...
司辰或Hours只不过是人类对于祂们的称呼，事实上祂们的权柄绝不仅限于时间，而是深入世界的方方面面。但祂们的确在大体上与一天中的特定时间段有关，尽管祂们在绝对数量上超过24个——“[圣坛](https://mansus.huijiwiki.com/wiki/%E5%9C%A3%E5%9D%9B)裁定了有三十位司辰”[1]。

# 司辰
...
司辰经常被描述为神，但祂们既不全知，也不全能。祂们也有旺盛的欲望（并且比任何人类可能拥有的欲望都要强烈得多），它们的渴求与激情对世界产生了极大的影响。[2]也有自身的缺陷，并且受到法则和其他司辰的掣肘。有时司辰甚至会被自身立下的法则束缚，而那些伟大的学者了解如何利用这些法则的漏洞达成自己的目的。[3]
